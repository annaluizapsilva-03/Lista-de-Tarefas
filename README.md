# Lista-de-Tarefas
Uma aplicação web simples desenvolvida com Python, HTML, CSS e JavaScript para gerenciar tarefas do dia a dia.

🎯 Objetivo

Este projeto foi desenvolvido como prática dos conteúdos estudados no primeiro semestre da graduação em Engenharia de Software, com foco em:

Lógica de programação
Desenvolvimento web 
Estruturação de aplicações
HTML, CSS e JavaScript
Organização de código

✨ Funcionalidades
Adicionar novas tarefas
Marcar tarefas como concluídas
Excluir tarefas
Interface simples e intuitiva
Layout responsivo

🛠️ Tecnologias Utilizadas
Python
Flask
HTML5
CSS3
JavaScript
